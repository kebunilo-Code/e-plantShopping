# coding-project-template
Project Name: e-plantShopping 
The goal of this program is to act as a rudementary shopping cart for the plants.
-   The program will provide a list of plants catogorized by diffrent features. 
-   Each item will contain an add to cart button
-   In the shopping cart will contain a list of all the seclected ites, with each item containg a increase (+) or decrease (-) button.
-   The user can also remove the item if the user wants to.
-   The bottom of the shoppping cart screen will also contain the total price of all of the items.
